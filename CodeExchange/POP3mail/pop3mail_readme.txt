README for POP3mail

Descriptions
------------

The source files enclosed in this zip files are:

POP3client.dbc
This is a POP3 class written in Synergy Language.

popmail.dbl
This file is a very simple pop3 client application that connects to a POP3
server, and shows a list of messages waiting.

POP3mail requires the SynPSG.System libraries available in ChronoTrack (the
SPC2009 sample application).
