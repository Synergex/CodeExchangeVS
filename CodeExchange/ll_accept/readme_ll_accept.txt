README.TXT for LL_ACCEPT.DBL

Description of function
-----------------------

This function will accept input from a Synergy low level window, and process
the input to return a single value which will be either the value of the
character entered or a value for function key pressed.  The function key
values are defined in KEYDEF.DEF
